ㅇ컴파일언어 : 번역기(인터프리터)가 있음 ex) 자바의 jvm
-> 에러가 있으면 컴파일이 불가능하여 에러발견이 쉬움(디버깅)
ㅇ스크립트언어(인터프리터언어) : 번역기 없음 
-> 한줄읽고 실행 한줄읽고 실행
-> 디버깅이 쉬움

  태그의 종류
  1. input류 : 사용자로부터 값을 입력받아 서버로 전달한다
  ex) input, select, textarea
  -> name속성과 value속성이 중요하다
  tag ex) <input type="text" name="userid" value="test">
    userid=value

***접근방법
  document.getElementsByName("userid");
  userid[0].value="test";
  
  2. 그 밖에(table, div, span)
  얘네들은 id속성과 class속성을 갖는다.
  -> 주의사항 : id는 무조건 문서 하나에 하나만 갖는다. 중복하면 안된다.
  tag ex)
  <span id="result">결과1</span>
  <span id="result">결과2</span>
  --> 화면에는 뜰 수 있으나 js에서 첫번째만 가져오고 두번째부터는 안가져와질 수 있다.

***접근방법
  var result = document.getElementById("result"); //첫번째것만 가져온다(중복이 안되므로 Elements가 아님에 주의)
  result.innerHTML = "새로운 결과";
